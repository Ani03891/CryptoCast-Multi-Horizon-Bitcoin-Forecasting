# CryptoCast: Multi-Horizon Bitcoin Price Forecasting Using Deep Learning

## Overview
CryptoCast is a deep learning-based time series forecasting project that predicts Bitcoin prices for multiple future horizons (1-Day, 3-Day, and 7-Day).

## Problem Statement
The objective is to develop and compare deep learning models to forecast Bitcoin prices using historical price data.

## Models Used
- SimpleRNN
- LSTM
- CNN
- Transformer

## Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow/Keras
- Scikit-learn
- Matplotlib

## Evaluation Metrics
Models were evaluated using:
- MAE
- RMSE
- R² Score

## Results
The LSTM model achieved the best performance among all models.

## Conclusion
LSTM was selected as the final model due to its ability to capture sequential dependencies in Bitcoin price data.
